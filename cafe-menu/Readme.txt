# Camper Cafe Menu

A simple and responsive cafe menu webpage built using HTML and CSS. This project displays a cafe menu featuring coffee and dessert items along with their prices in a clean and visually appealing layout.

## Overview

The webpage simulates a cafe menu for **Camper Cafe**, showcasing different beverages and desserts. It uses semantic HTML elements to organize content and CSS to create a polished design with a themed background, centered menu card, custom typography, and interactive links.

## Features

* Coffee and dessert menu sections
* Item names displayed alongside prices
* Responsive centered menu layout
* Coffee-themed background image
* Section icons for visual appeal
* Styled horizontal dividers
* Custom typography and spacing
* Footer containing website link and address
* Hover and active effects for links

## Project Structure

```text
index.html
styles.css
```

### index.html

The HTML file provides the structure of the webpage and contains:

* Main cafe heading
* Establishment year
* Coffee menu section
* Dessert menu section
* Menu items and prices
* Footer with website link and address

Semantic HTML elements such as `main`, `section`, `article`, `footer`, and `address` are used to improve readability and organization.

### styles.css

The CSS file is responsible for the visual design of the webpage and includes:

* Background image styling
* Font customization
* Menu container layout
* Text alignment
* Item and price positioning
* Image centering
* Divider styling
* Link state styling using pseudo-classes (`:visited`, `:hover`, `:active`)
* Responsive sizing with percentage widths and max-width constraints

## Key CSS Techniques Used

* Class selectors
* Grouped selectors
* Responsive design principles
* Auto margins for centering
* Inline-block display
* Width-based layout distribution
* Typography styling
* Background images
* Pseudo-classes for interactive elements

## Learning Objectives

This project was created to practice:

* Semantic HTML elements

* CSS selectors and classes
* Layout and spacing techniques
* Responsive webpage design
* Styling images and links
