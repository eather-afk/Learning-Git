# Business Card Webpage

## Overview

This project is a simple and responsive business card webpage created using HTML and CSS. It showcases basic front-end development concepts such as layout styling, typography, image responsiveness, link styling, pseudo-classes, and semantic HTML structure.

The webpage includes:

* Profile image
* Full name and designation
* Company/organization name
* Contact information
* Portfolio link
* Social media links

---

## Technologies Used

* HTML5
* CSS3

---

## Features

* Responsive profile image
* Center-aligned business card layout
* Custom typography using Arial font
* Hover, active, visited, and focus effects on links
* Styled social media section
* External links opening in a new tab

---

## Concepts Practiced

### HTML

* Semantic structure
* Div containers
* Paragraph elements
* Anchor tags
* Image elements
* Headings
* Horizontal rules

### CSS

* Class and ID selectors
* Font styling
* Background colors
* Margin and padding
* Text alignment
* Pseudo-classes:

  * `:hover`
  * `:active`
  * `:visited`
  * `:focus`
* Responsive image sizing
