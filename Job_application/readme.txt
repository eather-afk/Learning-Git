# Job Application Form

A responsive Job Application Form built using HTML and CSS.

## Preview

This project demonstrates:

* Semantic HTML structure
* Form validation styling
* Custom radio button effects
* Focus and hover states
* Responsive form container

## Features

* Responsive design
* Styled form inputs
* Input validation using CSS pseudo-classes
* Custom radio button selection styling
* Interactive button hover effects
* Accessible labels and form structure

## Technologies Used

* HTML5
* CSS3

## CSS Concepts Used

* `:focus`
* `:checked`
* `:valid`
* `:invalid`
* Adjacent sibling selectors
* Box shadows
* Border radius
* List selectors

## Project Structure

```bash
project-folder/
│
├── index.html
├── styles.css
└── README.md
```

## How to Run

1. Download or clone the repository
2. Open `index.html` in your browser

## Learning Outcomes

This project helped practice:

* Writing cleaner CSS selectors
