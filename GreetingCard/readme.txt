# Interactive Greeting Card

This project is a simple interactive greeting card built using HTML and CSS. It demonstrates the use of pseudo-classes, pseudo-elements, transitions, transforms, flexbox, and section targeting to create an engaging user experience.

## Features

* Responsive greeting card layout
* Hover animations using CSS transitions and transforms
* Interactive buttons with pseudo-classes
* Emoji decorations using pseudo-elements
* Hidden sections that appear when links are clicked
* Flexbox layout for button alignment
* Styled sections with hover effects

## Technologies Used

* HTML5
* CSS3

### HTML

* Semantic structure
* Anchor links

### CSS

* Flexbox

* Transitions

## How It Works

The project contains a greeting card with two interactive links:

* **Send Card**
* **Share on Social Media**

When a link is clicked:

* The browser targets the matching section using the URL hash (`#send` or `#share`)
* CSS `:target` selector makes the corresponding section visible
* All sections remain hidden by default until targeted
